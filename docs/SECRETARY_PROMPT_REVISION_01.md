## 修正 2: スライド 10「ショートカットキー一覧」の文字と背景のずれ

### 報告内容
前原さん目視確認で「**文字と背景がずれている**」とのこと。

### 想定される問題
- カード背景（Surface 色 RECTANGLE）の位置 / サイズと、その上のテキストボックスの位置 / サイズが一致していない
- 表のセル背景と文字が縦横方向にずれて見える
- アクセントバー or セパレーターの位置ずれ

### 修正方針
1. `slide_10.json` を `object_id_map.md` と照合しつつ確認
2. 各テキストボックス（左カラム/右カラム）の `transform.translateX/Y` と `size.width/height` を対応する背景シェイプの値と揃える
3. 背景シェイプ自体がずれていれば、デザインシステムの 8pt グリッド（X: 64, 80, 200 等 / Y: 144, 200, 250 等）に合わせて再配置
4. 修正後、サムネイル `slide_10.png` を再生成して目視確認

### 文言変更不要
ショートカットキー一覧の内容（スペース / [ / ] / ↑↓ / Ctrl+R / Ctrl+A / F11 / F12 / Ctrl+Q）は **変更不要**。位置 / サイズの整合のみ。
