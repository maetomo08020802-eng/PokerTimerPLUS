## プロンプト本文（ここからコピー）

PokerTimerPLUS+ という Windows デスクトップアプリの取説スライド一式を、PDF 配布用にゼロから作り直してほしい。

## プロジェクト概要

- **製品名**: PokerTimerPLUS+
- **業種**: ポーカートーナメントクロック（Electron 製 Windows アプリ）
- **配布**: 全国のポーカールーム向け、完全無料・完全ローカル動作
- **制作**: Yu Shimomachi（PLUS2 運営、戦略的フリーソフト）
- **戦略目的**: 「下町優」「PLUS2」ブランドの全国認知拡大
- **想定読者**: ポーカールーム運営者（多くがアプリ操作初心者）、業界プロから初心者まで
- **読者の閲覧シーン**: 配布された PDF をスマホ or PC で開いて読む

## デザイン要件（最重要）

**「超洗練された感じ」**を出してほしい。既存版（Python 自動生成、Premium ダーク + Gold アクセント）はあった、線がずれたり枠が画像に沿っていなかったりで満足できないため、**ガラッと方向性を変えたい**。

### 最初のステップ（必読）

いきなり全 12 枚作らず、まず**デザイン方向性 3 案のサンプル（タイトルスライドのみ 1 枚ずつ）**を提示してください:

1. **案 A: ミニマル・モダン**（Apple Keynote / Linear 風、白基調、大胆な余白、シャープなサンセリフ）
2. **案 B: ラグジュアリー・ダーク**（カジノ高級感、深いリッチブラック / 深緑 + プラチナ or ローズゴールドアクセント、シックで重厚）
3. **案 C: スイス・タイポグラフィ**（グリッドベース、白黒 + 1 アクセント色、最高峰の洗練、文字が主役）

3 案見せたら、私が「これでいく」と方向性を選ぶので、選んだ方向性で残り 11 枚を量産してください。

### 共通要件（どの方向性でも厳守）

- **日本語フォント**: 高品質な和文フォント（Noto Sans JP / BIZ UDPGothic / Inter Tight + Noto Sans JP 等）
- **数字フォント**: 既存のアイデンティティ尊重で **Barlow Condensed 700**（タイマー数字専用）も使えるが、必須ではない
- **画像の扱い**: アスペクト比保持、枠と画像のずれをゼロに（CSS の `object-fit: cover` 相当）
- **アクセシビリティ**: コントラスト比 WCAG AA 以上
- **PDF 配布前提**: 1 枚 1 ページで配布されるため、各スライド単独で完結する情報量
- **アスペクト比**: 16:9（標準プレゼン比率）

### ブランド要素（厳守）

- 制作者: **Yu Shimomachi**（タイトル / クレジットに必ず明記）
- 連絡先 or リポジトリ URL: <https://github.com/maetomo08020802-eng/PokerTimerPLUS>
- バージョン: **v2.0.6**
- 配布形態: フリー配布（完全無料）

## スライド構成（全 12 枚）

各スライドのタイトル + 本文 + 画像配置を以下に詳細指示する。

### スライド 1: タイトル

- **メインタイトル**: PokerTimerPLUS+
- **サブタイトル**: 全国のポーカールームへ、無料配布のトーナメントクロック
- **バージョン**: v2.0.6
- **制作**: presented by Yu Shimomachi
- **配置**: 最も洗練された印象を与えるスライドにすること

### スライド 2: 起動時の警告について

- **タイトル**: 初回起動時の Microsoft 警告について
- **本文**:
  - PokerTimerPLUS+ はコードサイニングなしで配布されています（コスト削減）
  - 初回起動時に Windows の SmartScreen 警告（青い画面）が出る場合があります
  - 「**詳細情報**」をクリック → 「**実行**」を押せば起動します
  - 位置情報・カメラ・マイクなど一切使用していません（完全ローカル動作）

### スライド 3: 概要（このアプリでできること）

以下の 9 機能を 2 カラムレイアウトで:

1. **ブラインドタイマー** — カウントダウン / 警告音 / レベル終了音 / スタート前カウントダウン
2. **多彩なゲーム種対応** — NLH / PLO / PLO8 / Limit / Stud / Short Deck / 10-Game MIX / 自由記入で独自ゲーム種登録可
3. **複数トーナメントの並行管理 + 店名表示** — タブ切替で同時運営、画面下部に店舗名常時表示
4. **賞金プール / PRIZE 設定** — 自動計算 + GTD 保証額対応 + 順位別ペイアウト表示
5. **特殊状況の +スタック対応** — リエントリー / アドオン / リバイで人数とスタックを柔軟管理
6. **背景設定 8 種類 + カスタム画像** — 黒・紺・カーボン・フェルト等 + 任意画像（暗くする強度 3 段階）
7. **休憩中の告知スライドショー** — 複数画像で告知 / 30 秒遅延・1 分前自動復帰 / 右下に縮小タイマー
8. **自由記入テロップ + サウンド選択** — 流れる文字を自由設定 / カウント音を試聴して選択
9. **PC 間データ移行** — トーナメント設定を Export / Import で他 PC へ持ち運び

### スライド 4: 画面の見方

メインのタイマー画面（単画面モード時）の構成要素を解説:
- 上部: イベント名（カスタム可能）
- 中央: 大きなタイマー数字（残り時間）
- 左右: 現在ブラインド / 次ブラインド カード
- 下部: ステータスバー（プレイヤー数 / 平均スタック / 賞金 / リエントリー数 等）+ 横スクロールテロップ
- フッター: 制作者表記 + 店名

**末尾に注記**: ※ HDMI 接続で 2 画面で運用する場合は次のページ参照

### スライド 5: 2 画面で運用する（HDMI 接続）★新規・最重要

HDMI ケーブルで会場モニターを繋ぐと、自動的に「会場側のタイマー画面」と「手元 PC の操作画面」を分けて表示できる機能の説明。

3 ステップ + 3 画像（後述で画像パスを指定）:

| ステップ | 見出し | 説明 | 画像 |
|---|---|---|---|
| 1 | **準備：PC を「拡張」表示にする** | HDMI ケーブルで会場モニターを PC に接続 → Windows キー + P を押して表示モードを「拡張」に切り替え（複製や PC 画面のみではなく拡張） | `01-display-extend.png` |
| 2 | **アプリ起動：モニターを選ぶ** | アプリを起動すると「ホール側のモニターを選択」ダイアログが出る → お客様に見せる側のモニターを選択ボタンで指定。「キャンセル」を押すと従来の単画面モードで起動 | `02-monitor-select.png` |
| 3 | **手元 PC で操作** | 会場モニター側はタイマーが全画面表示（単画面モードと同じ表示）、手元 PC 側には運用情報 + 操作一覧が表示される。HDMI を抜くと自動的に単画面モードに切り替わる（タイマーは途切れず継続） | `03-operator-pane.png`（手元 PC 画面）|

**補足**: HDMI ケーブルが繋がっていない PC ではこのダイアログは出ない（自動的に単画面モードで起動）。HDMI を抜き差しすると自動的にレイアウトが切り替わる（v2.0.4 以降の機能）。

### スライド 6: トーナメント設定

- 新規トーナメント作成 / 複製 / 削除
- イベント名・人数・初期スタック・スタート前カウントダウン秒数の設定
- 14 種類のゲーム種から選択（NLH / PLO / Stud / WSOP 10-Game MIX / その他自由記入 等）
- 保存して適用 / 構造のみ適用 / リセット の 3 択（一時停止中）

### スライド 7: ブラインド設定

- 同梱 7 プリセット（ターボ / レギュラー / ディープ / Stud Regular / Short Deck Regular / Limit Regular / WSOP 10-Game MIX）
- プリセットを「複製して編集」で自由カスタマイズ
- レベルごとに SB / BB / アンティ / 時間 / ブレイクの自由設定
- v2.0.5 で同梱プリセットの値を業界標準値に更新済（1500/2000/2500/3000/4000... 系列）

### スライド 8: 背景・表示設定

- 9 種類の背景プリセット: 黒 / 紺 / カーボン / フェルト / 緋 / 真夜中 / エメラルド / オブシディアン + カスタム画像
- カスタム画像は暗くする overlay 自動適用（強度: low / mid / high の 3 段階）
- フォント設定（タイマー数字フォント、イベントタイトルフォント）
- 表示設定（テロップ太さ、PIP タイマーサイズ）

### スライド 9: 休憩中スライドショー設定

- 最大 20 枚の画像を登録可能（5MB/枚、PNG/JPEG/WebP）
- 切替間隔: 3〜60 秒の数値入力
- PIP（小窓）タイマーサイズ: 小 / 中 / 大
- 起動時 150MB 超で警告
- 1 枚しか登録されていない場合は静止表示（点滅しない、v2.0.6 改善）
- 切替間隔の変更は実行中のスライドショーにも即時反映（v2.0.6 修正）

### スライド 10: 休憩中スライドショー実行時の動作

- 休憩開始 30 秒後にスライドショー開始
- タイマーは右下に PIP（小窓）表示で継続
- 自動切替（指定秒数間隔）+ クロスフェード
- スタート 1 分前に自動でフルスクリーンタイマーに復帰
- 手動で「タイマーに戻る」「スライドショーに戻る」ボタンで切替可能
- 「スライドショーに戻る」ボタンは画面左の縦中央付近に配置（v2.0.6 改善、テロップに隠れない位置）
