## 🖥️ 新機能: マルチ表示モード（4分割）

大型大会向けの新モードを追加しました。**自動更新で順次配信されます。単画面・従来の 1 トーナメント表示のご利用には影響ありません（これまでどおり動きます）。**

### 最大 4 つのトーナメント時計を同時表示

- 設定画面の「ハウス情報」タブから開始すると、巨大スクリーン（HDMI 拡張モニター）に**最大 4 つのトーナメント時計を 2×2 で同時表示**できます。
- 各区画は**完全に独立した時計**です。開始時刻・ブラインド構造・一時停止・レベル進行がそれぞれバラバラに動きます。
- 空いている区画には「無地 / 店舗ロゴ / 任意の画像 / 固定テキスト」を表示できます。

### 手元 PC の操作盤 + キーボード操作

- 区画ごとに **スタート / 一時停止 / レベル送り・戻し / リセット / ±30秒 / エントリー・脱落・リエントリー・アドオン** を操作できます。
- モニター複製（ミラーリング）で使う場合は**キーボードだけでも操作**できます（1〜4 キーで区画選択、H キーで操作一覧）。

### トラブルに強い設計

- **停電・強制終了からの復帰**: 不意にアプリが落ちても、次回開始時に直前の状態を復元できます。「そこから再開（時計は進めない）」か「経過を反映して復元（停電中の時間も進んだ扱い）」かを選べます。
- **HDMI 抜き差しに自動対応**: ケーブルが抜けてもタイマーは動き続け、挿し直すと表示が自動復帰します。
- **誤操作ガード**: 進行中の区画をうっかり差し替えてもタイマーや人数が消えないよう、確認画面を挟みます。
- **終了時の確認**: 進行中の区画があるまま × ボタンなどで終了しようとすると「元に戻せません」の確認が出ます（うっかり終了で進行が消えるのを防止）。

### 大切なデータはそのまま

- マルチ表示モードは登録済みトーナメントを**読み込むだけ**です。このモードでの操作が登録データに保存・上書きされることはありません。
