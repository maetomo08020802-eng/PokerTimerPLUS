# PokerTimerPLUS+ 配布版 動作確認チェックリスト

ビルド完了後、配布前に必ず一通り確認してください。
クリーンな Windows 環境（または別アカウント）で確認するのが理想ですが、開発機での確認でも十分です。

## インストール
- [ ] `dist\PokerTimerPLUS+ Setup 1.0.0.exe` をダブルクリックで起動
- [ ] **Windows SmartScreen 警告** が出る（コードサイニングなしのため）→「詳細情報」→「実行」で続行
- [ ] インストール先選択画面が出る（任意のパスにインストール可能）
- [ ] インストール完了 → スタートメニューに「PokerTimerPLUS+」が追加される
- [ ] デスクトップショートカット（あれば）も「PokerTimerPLUS+」表示

## 起動
- [ ] スタートメニューから「PokerTimerPLUS+」起動
- [ ] **タイトルバー**: 「PokerTimerPLUS+ — presented by Yu Shitamachi」表示
- [ ] **アプリアイコン**: タスクバー / タイトルバーに PLUS2 ロゴ
- [ ] メイン画面（タイマー / 左カラム / 右カラム）が正しく表示
- [ ] フォントが正しく表示される（Inter / JetBrains Mono / Noto Sans JP 同梱、CDN 不要）

## 機能チェック（5〜10 分）

### タイマー基本
- [ ] スタートボタン → カウントダウン開始
- [ ] 一時停止 → 残り時間が固定
- [ ] 再開 → 続きからカウントダウン
- [ ] リセット → レベル 1 / 残り時間初期化
- [ ] 5秒前カウント / レベル境界音 / ブレイク音 が鳴る

### ショートカット
- [ ] `Space`: スタート/一時停止トグル
- [ ] `←` / `→`: ±30秒
- [ ] `R`: リセット確認
- [ ] `F11`: フルスクリーン
- [ ] `F12`: DevTools（コードサイニングなしでも開ける）
- [ ] `Ctrl+T`: テロップ編集ダイアログ
- [ ] `S`: 設定ダイアログ

### トーナメント
- [ ] 新規 / 複製して編集 / 各行🗑削除（最後の1件は削除不可）
- [ ] 名前 / サブタイトル / バイイン / 賞金構造 / 特殊スタック の編集 + 保存
- [ ] active 切替で背景・フォント・テロップが各トーナメントの設定に切替
- [ ] 連番命名（「新規トーナメント」「新規トーナメント 2」「新規トーナメント 3」）

### ブラインド構造
- [ ] 同梱4プリセット（demo-fast / preset-turbo / preset-regular / preset-deep）切替
- [ ] 同梱選択時に「📋 このプリセットは元のまま編集できません...」金色警告表示
- [ ] 「複製して編集」で新規ユーザープリセット作成
- [ ] 「適用」3択モーダル（リセット適用 / 経過時間保持 / キャンセル）

### 設定タブ
- [ ] 「トーナメント」「ブラインド構造」「背景・時計フォント」「テロップ」「音」「ハウス情報」の 6 タブ
- [ ] 背景プリセット 8 種 + 数字フォント 3 種が切替可能
- [ ] テロップ編集 + プレビュー（閉じると保存値に復元）
- [ ] 音タブ: マスター音量 + 各音 ON/OFF + 音色バリアント
- [ ] **ハウス情報** → 店舗名入力 →「保存しました」緑字 → メイン画面に「Presented by ○○」表示
  - 不正入力（日本語、先頭記号）→ 赤字エラー
  - 9文字超で見切れ注記表示

### データ移行
- [ ] エクスポート（単体ファイル / 単体クリップボード / 一括ファイル）
- [ ] インポート（ファイル / クリップボード）+ 戦略選択（別名追加 / 上書き / Esc キャンセル）
- [ ] 不正 JSON / 古いバージョンの拒否

### 停電・スリープ復元（任意、時間あれば）
- [ ] 強制終了 → 5秒後再起動 → タイマーが進んだ状態で復元（巻き戻らない）
- [ ] running 中に PC スリープ → 復帰 → タイマーが進んだ状態で再同期

## アンインストール（最後に）
- [ ] スタートメニュー → アプリ右クリック → アンインストール（または「設定 > アプリと機能」から）
- [ ] アンインストール完了
- [ ] **`%APPDATA%\PokerTimerPLUS+\config.json` は残っている**
  - 確認方法: エクスプローラーのアドレスバーに `%APPDATA%\PokerTimerPLUS+` を入力
  - `deleteAppDataOnUninstall: false` の効き目確認
  - 再インストール時にトーナメントデータが復元されることを担保

## 不具合があれば
- スクリーンショット + DevTools コンソール（F12 → Console タブ）のエラーメッセージを CC構築士に共有
- 再現手順を箇条書きで添える（操作 1 → 2 → 3 → 期待 vs 実際）
